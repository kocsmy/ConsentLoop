Everything a CMP does. None of the weight.

The banner asking permission to load scripts shouldn’t be the heaviest script on your page.

Performance first

A ~1 KB loader handles returning visitors without ever downloading the UI. The full widget renders after first paint, off the critical path.

Zero layout shift

Every surface is an overlay — your content never jumps. Animations use transform and opacity only, and respect reduced motion.

Script & embed gating

Block scripts, iframes and pixels declaratively with data-consent. YouTube embeds wait politely for marketing consent.

Consent Mode v2

Correct gtag('consent') defaults and updates out of the box, mapped from your categories. Google tags just work.

GDPR & CCPA presets

Opt-in for Europe, opt-out for California — one regulation switch. Revisions re-prompt everyone when your policy changes.

AI-native docs

llms.txt, a JSON schema, strict TypeScript types and copy-paste prompts. Claude, Cursor & Copilot integrate it in one shot.

Add it in 30 seconds

Use the CDN two-liner, the npm package, or React bindings. The banner on this page is ConsentLoop itself — .

<script defer src="https://cdn.jsdelivr.net/npm/consentloop/dist/consentloop.loader.min.js" data-gcm></script>

<!-- gate anything on consent -->
<script type="text/plain" data-consent="analytics" data-consent-src="https://plausible.io/js/script.js"></script>
<iframe data-consent="marketing" data-consent-src="https://www.youtube.com/embed/dQw4w9WgXcQ"></iframe>
import * as ConsentLoop from "consentloop";

ConsentLoop.run({
  categories: {
    necessary: { required: true },
    analytics: {},
    marketing: {},
  },
  googleConsentMode: true,
});
import { ConsentProvider, ConsentGate, useConsent } from "@consentloop/react";

export default function App() {
  return (
    <ConsentProvider config={{ categories: { analytics: {}, marketing: {} } }}>
      <Site />
      <ConsentGate category="marketing" fallback={<Placeholder />}>
        <YouTubeEmbed id="dQw4w9WgXcQ" />
      </ConsentGate>
    </ConsentProvider>
  );
}
